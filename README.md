# 8051-Digital-Voting-Machine
A 2-digit voting system with winner and tie logic using AT89S52

Digital Voting Machine (8051-Based)

Student: Syed Shafqatullah Shah

Registration No: 24PWELE6297

**Project Overview**

This is a 4th-semester Electrical Engineering project involving an AT89S52 microcontroller and an LCD1602 display. It supports up to 99 votes for three candidates and handles complex tie-breaking logic.

**Features**

1). 2-digit display support (00-99).

2). Comprehensive result calculation (Winner/Tie/No Votes).

3). Master Reset functionality.
